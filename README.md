# Installation

- clone this repository,
- run command ```
npm install ```

# Running dev

```cmd
npm run dev
```


# API Reference

- GET ``` /api/users/ ``` get the users data
- POST ``` /api/users/login/``` user login auth
- POST ``` /api/users/ ``` adding a new user data
- PUT ``` /api/users/edit?id=1 ``` edit data user
